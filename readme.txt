A front-end for the Composer package manager to search and install packages.

== Description ==

Edit and update your Composer project configuration!

Features:

*   Editor for composer.json.
*   Search packages.
*   List installed packages.
*   Run Composer update.
*   Configure Composer.

== Installation ==

1. Install The Blackest Box's Composer either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Install Redux Framework either via the WordPress.org plugin directory, or by uploading the files to your server.
3. Activate both plugins.
4. Change your composer.json path and composer.phar in the settings.
5. That's it.  You're ready to go!
